Layout (лэйаут)

Layout (лэйаут) - расположение элементов и блоков относительно друг друга в рамках страницы или интерфейса.

В Laravel встроен шаблонизатор Blade, который позволяет настривать layout страницы по блокам, при этом каждый блок может быть переопределен.

Layout (лэйаут)

Blade, Layout

Для натсройки layout необходимо определить основные секции страницы, и вынести их в отдельные файлы, например меню, шапка, внутренний блок контента, футер, и т.д.
В каждой секции, если необходимо переопределять данные, можно использовать директивы @yield или @section.

Blade, Layout

В дочерних шаблонах переопределение данных выполняется с помощью директивы @section(NAME) ... @endsection
